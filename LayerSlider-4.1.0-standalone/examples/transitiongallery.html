Example 7: 3D & 2D transition gallery

You can see all the available 3D & 2D transitions in the list below. Please move your mouse over the names of the transitions and you will see a real-time preview of the selected transition. Please note, that not all browsers are compatible with CSS3 3D transitions. You can use 3D transitions with the latest version of Chrome (except on Win XP), Firefox, Safari, and soon with Opera, and of course with the CSS3 3D capable mobile browsers. In all other browsers the slider will switch into a 2D fallback mode. LayerSlider has very nice 2D transitions as well :)
